判断基準チェックリスト

🤔 このエージェントを選ぶ前に確認：
  • 対応プラットフォーム（Linux / macOS / Windows）は？
  • API キー設定は必要か？ → SaaS か Local か
  • リアルタイム反応が必要か？ → IDE 系か CLI 系か
  • Git 統合が必須か？ → aider / Goose を検討
  • 画像認識が必要か？ → Gemini / Antigravity
  • ローカル・プライバシー重視か？ → OpenCode / Local 系
  • 予算制約（API 費用）はあるか？
