Claude Code
Anthropic
会話型 CLI Agent
対話形式で実装やデバッグを段階的に進められる CLI エージェントです。
できること
  • 会話型インターフェース (REPL)
  • マルチステップ推論・計画立案
  • ファイル読み書き・git操作
  • Shell実行＋結果フィードバック
ユースケース
  • 設計・実装フローの相談
  • コードレビュー＆リファクタリング
  • バグ修正の思考プロセス
  • 複雑なロジックの説明
CLI REST API Git
↗ claude.ai/code

Windsurf
Codeium
GUI 統合 IDE
IDE 内で AI 補完やコード生成を直接利用できる統合開発環境です。
できること
  • VSCode 互換（Extensions 活用可）
  • インライン AI 補完・コード生成
  • Context-aware なサジェスト
  • 統合ターミナル＆デバッグ
ユースケース
  • 日常開発・タイピング時短
  • テンプレート・定型コード生成
  • IDE のネイティブ体験が必要な場合
  • リアルタイムフィードバック
IDE GUI Git
